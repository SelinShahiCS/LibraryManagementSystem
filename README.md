# Library Management System (C++)

A console-based Library Management System developed in C++ to demonstrate Object-Oriented Programming (OOP) concepts and file handling.

## Features

### Admin
- Login / Logout
- Add and delete books
- Search books by title or author
- View all books

### User
- Sign up / Login
- View available books
- Borrow and return books

## OOP Concepts
- Inheritance
- Encapsulation
- Constructors & Destructors
- Friend Class

## File Storage
- `books.txt`
- `users.txt`

## Technologies
- C++
- g++
- File Handling (TXT)


## Future Improvements
